## Висновки

Для обчислення інтеграла функції за допомогою методу Монте-Карло було згенеровано велику кількість випадкових точок всередині прямокутника, обмеженого межами інтегрування та графіком функції. Після цього було пораховано кількість точок, які потрапили під криву, та відношення цієї кількості до загальної кількості точок, що дало наближене значення інтеграла.

Порівнюючи результат обчислення методом Монте-Карло з аналітичним розрахунком (за допомогою функції `quad` з бібліотеки SciPy), було виявлено, що отримані значення співпадають у межах прийнятної похибки. Це підтверджує точність методу Монте-Карло для обчислення інтегралів.

Отже, метод Монте-Карло є ефективним інструментом для наближеного обчислення інтегралів, особливо в складних випадках, коли аналітичні розрахунки можуть бути важкими або неможливими.
